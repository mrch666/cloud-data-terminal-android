# ОТЧЕТ О ВЫПОЛНЕНИИ ПРОЕКТА: CLOUD DATA TERMINAL ANDROID

## 📋 ОБЗОР ПРОЕКТА
**Cloud Data Terminal Android** - мобильное приложение для сканирования штрих-кодов товаров и управления инвентарем с облачной синхронизацией. Аналог Cloud Data Terminal с сайта https://wiki.lineris.ru/cloud_data_terminal.

## 🎯 ВЫПОЛНЕННЫЕ ЗАДАЧИ

1. **Создание структуры проекта** - Clean Architecture + MVVM
2. **Настройка Gradle и зависимостей** - Kotlin, Jetpack Compose, CameraX, ML Kit
3. **Создание основных компонентов** - Application, MainActivity, навигация

## 🏗️ АРХИТЕКТУРА ПРОЕКТА

### Структура проекта (Clean Architecture)
```
app/
├── src/main/java/com/cloudterminal/
│   ├── presentation/          # UI слой (Jetpack Compose)
│   │   ├── screens/          # Экранные композейблы
│   │   ├── components/       # Переиспользуемые компоненты
│   │   ├── navigation/       # Навигация
│   │   └── theme/           # Тема приложения
│   ├── domain/               # Бизнес-логика
│   │   ├── models/          # Domain модели
│   │   ├── repository/      # Интерфейсы репозиториев
│   │   └── usecases/        # Use Cases
│   ├── data/                # Data слой
│   │   ├── local/           # Локальное хранение (Room)
│   │   ├── remote/          # Удаленные данные (Retrofit)
│   │   └── repository/      # Реализации репозиториев
│   └── di/                  # Dependency Injection (Hilt)
└── src/main/res/            # Ресурсы приложения
```

### Технологический стек
- **Язык**: Kotlin 1.9.0
- **UI**: Jetpack Compose 1.5.0 + Material Design 3
- **Камера**: CameraX 1.3.0
- **Штрих-коды**: ML Kit Barcode Scanning 17.2.0
- **База данных**: Room 2.5.2
- **Сеть**: Retrofit 2.9.0 + OkHttp
- **DI**: Dagger Hilt 2.48
- **Асинхронность**: Kotlin Coroutines + Flow
- **Тестирование**: JUnit 5, Mockito, Compose Testing

## 🚀 ИНСТРУКЦИЯ ПО ЗАПУСКУ

### Быстрый старт
1. **Клонировать репозиторий**:
   ```bash
   git clone https://github.com/mrch666/cloud-data-terminal-android.git
   ```

2. **Открыть в Android Studio**:
   - Android Studio Flamingo или выше
   - Android SDK 34
   - Java 17

3. **Собрать и запустить**:
   - Дождаться синхронизации Gradle
   - Нажать Run (Shift+F10)
   - Выбрать эмулятор или устройство

### Альтернативная сборка
```bash
# Сборка debug APK
./gradlew assembleDebug

# Сборка release APK
./gradlew assembleRelease
```
