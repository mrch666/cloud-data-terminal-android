# 🗺️ Roadmap проекта

## 🎯 Цели проекта

Создать современное Android приложение для сканирования штрих-кодов товаров с облачной синхронизацией, которое станет мобильной альтернативой существующему Cloud Data Terminal.

**Статус:** ✅ Проектная структура создана

**Что сделано:**
- [x] Создана полная структура проекта
