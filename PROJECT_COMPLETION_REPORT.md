## 🚀 ИНСТРУКЦИЯ ПО ЗАПУСКУ

### Быстрый старт (для разработчиков)
```bash
# 1. Клонировать репозиторий
git clone https://github.com/mrch666/cloud-data-terminal-android.git

# 2. Открыть в Android Studio
#    - Android Studio Flamingo или выше
#    - Android SDK 34
#    - Java 17

# 3. Собрать и запустить
./gradlew assembleDebug
```

### Для конечных пользователей
1. Собрать APK в Android Studio
2. Установить на Android устройство (7.0+)
3. Разрешить установку из неизвестных источников
4. Запустить приложение
